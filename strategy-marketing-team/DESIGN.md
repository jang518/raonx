# DESIGN.md — 오브온(OBON) 디자인 기준

> 목적: Codex가 대시보드, 보고서, 프로모션 이미지 시안을 만들 때 참고할 오브온 브랜드 디자인 기준입니다. `DESIGN-SAMPLE.md`의 구조를 참고했지만, 내용은 오브온 예시 브랜드에 맞춰 한국어로 재정리했습니다.

## 1. 브랜드 인상

오브온은 룸스프레이, 디퓨저, 선물세트를 판매하는 작은 D2C 라이프스타일 브랜드입니다. 핵심 이미지는 “잠들기 전 10분 루틴”, “집 안의 조용한 전환”, “과하지 않은 선물”입니다.

브랜드는 고급 호텔처럼 차갑게 보이기보다, 집 안의 따뜻한 조명과 깨끗한 침구, 나무 선반, 세라믹 오브제가 떠오르는 방향으로 갑니다. 과장된 럭셔리보다 차분하고 현실적인 프리미엄입니다.

## 2. 디자인 키워드

- 차분함: 여백이 넓고 정보가 과밀하지 않다.
- 따뜻함: 순백보다 아이보리, 검정보다 차콜을 쓴다.
- 루틴: 사용 장면, 시간대, 기분 전환을 보여준다.
- 신뢰감: 숫자와 근거는 명확하게, 장식은 절제한다.
- 선물감: 패키지와 카드, 작은 리본 같은 디테일을 적게 사용한다.

## 3. 색상 토큰

### 기본 색

- `canvas` — #FAF7F2: 기본 배경. 따뜻한 아이보리.
- `surface` — #FFFFFF: 카드, 표, 대시보드 패널.
- `ink` — #25221D: 제목과 핵심 텍스트. 순검정 금지.
- `body` — #4B453C: 본문 텍스트.
- `muted` — #7A7065: 보조 설명, 캡션, 날짜.
- `hairline` — #E5DDD2: 카드 경계, 표 구분선.

### 브랜드 포인트

- `sage` — #7E8F74: 안정, 루틴, 재구매 의향 같은 긍정 지표.
- `clay` — #C97A54: 주목 CTA, 다음 달 액션, 강조 배지.
- `sand` — #E8D8C5: 부드러운 배경 강조.
- `night` — #39414A: 저녁 루틴, 리스크, 깊은 정보 카드.
- `mist` — #EEF1ED: 보조 배경, 차트 보조 영역.

### 데이터 시각화 색

- 매출: `clay` #C97A54
- 만족도: `sage` #7E8F74
- 재구매 의향: `night` #39414A
- 인스타 저장/공유: #B58B62
- 리스크/주의: #B85C5C

## 4. 타이포그래피

- 기본 폰트: Pretendard, Apple SD Gothic Neo, Noto Sans KR, sans-serif
- 제목: 28–36px, 700, 자간 -0.02em
- 섹션 제목: 20–24px, 700
- 본문: 15–16px, 400–500, line-height 1.55
- 캡션/메타: 12–13px, 400, muted 색상
- 숫자 강조: 32–48px, 700, line-height 1.1

문장은 너무 딱딱한 SaaS 톤보다 “대표가 바로 판단할 수 있는 보고서” 느낌으로 씁니다. 단, 감성 카피와 데이터 설명을 섞지 않습니다. 숫자는 숫자답게, 메시지는 메시지답게 분리합니다.

## 5. 레이아웃 원칙

- 대시보드는 12컬럼 그리드 또는 2단 카드 레이아웃을 기본으로 한다.
- 주요 KPI 카드는 첫 화면 상단에 3–4개만 둔다.
- 차트보다 “그래서 무엇을 할지” 액션 카드가 더 중요하다.
- 여백은 넉넉하게 둔다. 카드 간격 20–24px, 섹션 간격 40–56px.
- 모바일 대응 시 KPI 카드 → 핵심 결론 → 차트 → 액션 순서로 쌓는다.

## 6. 컴포넌트 기준

### 카드
- 배경: `surface`
- 테두리: 1px `hairline`
- 라운드: 18px
- 그림자: 아주 약하게만 사용. `0 8px 24px rgba(37, 34, 29, 0.06)` 이하.

### 버튼
- 기본 CTA: `clay` 배경 + 흰색 텍스트
- 보조 버튼: 투명 배경 + `ink` 텍스트 + 1px `hairline`
- 라운드: 999px 또는 12px
- 버튼 문구는 짧게: “다음 달 액션 보기”, “보고서 열기”

### 배지
- 긍정: `sage` 12% 배경 + `sage` 텍스트
- 주의: 리스크 컬러 10% 배경 + 리스크 텍스트
- 정보: `mist` 배경 + `night` 텍스트

### 차트
- 막대/라인은 3색 이하.
- 범례는 차트 위가 아니라 차트 아래 또는 오른쪽에 작게.
- 차트 제목은 “매출 추이”보다 “선물세트는 매출은 높지만 환불도 높음”처럼 해석형으로 쓴다.

## 7. 이미지/프로모션 시안 기준

- 장면: 침대 옆 협탁, 욕실 선반, 현관, 선물 포장, 저녁 조명.
- 톤: 자연광, 따뜻한 그림자, 저채도 베이지/세이지/우드.
- 피해야 할 것: 과한 향수 광고 느낌, 금색 장식 남발, 실제 유명 브랜드 로고, 의료/수면 개선 과장.
- 이미지 문구 예시:
  - “잠들기 전 10분, 방의 공기를 바꾸는 시간”
  - “향보다 먼저 오는 기분 전환”
  - “선물세트에 루틴을 담다”

## 8. 대시보드 출력물 기준

대시보드는 예쁘기보다 의사결정이 빨라야 합니다.

첫 화면에서 보여줄 것:
1. 이번 달 가장 많이 팔린 상품
2. 만족도와 재구매 의향이 높은 상품
3. 저장/공유 반응이 좋은 콘텐츠 메시지
4. 다음 달 액션 우선순위 3개
5. 데이터 누락/추정 금지 경고

모든 결론에는 “근거 데이터”와 “다음 행동”이 같이 있어야 합니다.

## 9. 금지 사항

- 보이지 않는 구매 전환을 추정해서 쓰지 않는다.
- 실제 브랜드 로고나 유명 브랜드의 시각 정체성을 복제하지 않는다.
- “수면 개선”, “스트레스 치료”처럼 의학적 효능으로 읽힐 수 있는 표현을 쓰지 않는다.
- 감성 문구만 있고 숫자 근거가 없는 보고서를 만들지 않는다.
- 차트를 많이 넣는 대신 결정이 안 보이게 만들지 않는다.
